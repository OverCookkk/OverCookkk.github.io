Github网页上图片显示失败的问题

发表于2023-11-23|更新于2023-11-23|优雅写作图床搭建
|阅读量:

问题

比如在github上打开一个项目，项目里的图片无法查看，导致在博客显示的图片也显示不了，点击F12打开控制台查看

报错显示连接超时等问题。

解决方法

主要思路就是使用本地hosts文件对网站进行域名解析，一般的DNS问题都可以通过修改hosts文件来解决，

找到URL

通过网页的控制台，看到图片对应的URL为https://raw.githubusercontent.com/OverCookkk/PicBed/master/blogImg/git%20img1.png

获取IP地址
